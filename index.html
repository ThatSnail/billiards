Billiard Balls

A fully reversible computational model, really nice for demonstrating conservative logic.



Up - Zoom In
Down - Zoom Out
Q - Vertical Wall
W - Horizontal Wall
E - Enable / Disable
R - Remove
A - Argument
S - Result
D - Source
F - Sink
U - Run
I - Start
O - Stop
P - Pause
J - Step Forward
K - Step Backward (currently broken!)
L - Load
T - Add Text
